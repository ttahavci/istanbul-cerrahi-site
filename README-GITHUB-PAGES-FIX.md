# GitHub Pages Görsel ve Yol Düzeltmesi

Bu paket, proje GitHub Pages üzerinde `/istanbul-cerrahi-site/` alt yolu ile çalışırken görsellerin ve iç bağlantıların doğru okunması için hazırlanmıştır.

## Yapılan düzeltmeler

- Tüm görsel yolları `assets/img/` klasöründen okunacak şekilde korunmuştur.
- GitHub Pages proje yolu için bağlantılar `/istanbul-cerrahi-site/` tabanına göre güncellenmiştir.
- `.nojekyll` dosyası eklenmiştir.
- Görseller `assets/img/` altında kategori kategori tutulmaktadır.

## Yükleme yöntemi

GitHub web arayüzünde dosya limiti oluşabileceği için GitHub Desktop ile mevcut repo klasörünün üzerine bu paket içeriği kopyalanıp commit/push yapılması önerilir.
